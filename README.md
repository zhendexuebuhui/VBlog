# VBlog 个人博客管理系统

### 技术栈：SpringBoot + SpringMVC + JPA，LayUI，MySql 

### 技术特点：

1. 登录验证码

2. Echarts 数据图表

3. resultful 风格接口，易于扩展维护

4. Maven 进行项目管理

### 功能：

1. 文章管理

2. 新闻管理

3. 博客主页轮播组图管理

4. 个人信息管理

5. 文章新闻点击数、阅读数排行
